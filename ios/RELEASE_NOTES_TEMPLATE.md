# Release Notes Template

## Version 1.0.0

### What's New
- New native Snoke iOS app with onboarding and secure session setup.
- Dashboard with craving risk, daily targets, and quick logging actions.
- History screen with a 7-day trend chart.
- Configurable backend URL in Settings.
- Offline fallback showing last synced data when connection is unavailable.
- Optional daily reminder notifications for check-ins.

### Bug Fixes
- Improved handling for temporary backend errors.
- Better empty states for first-time users.

### Notes
- You can update the backend endpoint from in-app Settings.
